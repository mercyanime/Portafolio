# Portafolio de Diseño Gráfico 🎨

Portafolio personal creado con HTML, Tailwind CSS y JavaScript. Diseño limpio, moderno y totalmente responsivo.

## 🚀 Características
- Diseño adaptable a móviles, tabletas y escritorio
- Secciones: Inicio, Portafolio, Sobre mí, Servicios y Contacto
- Efectos visuales y animaciones suaves
- Código abierto y fácil de personalizar
